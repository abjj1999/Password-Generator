# Password Generator Starter Code


# Description
 A full stack website that generates a random and unique password.
 The user have the ability to choose what to include in the password.


 # How it works
 - The website will prompt the user to choose the length of the password.
 - The user will be then prompted to choose to include lower case characters, upper case characters, and/or special characters.
